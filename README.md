# Prompt Stocker 🗃️

生成画像からプロンプトを探せる「ビジュアル検索型」プロンプト管理ツール

## 特徴

- 📸 **ビジュアル検索**: 画像サムネイルからプロンプトを探せる
- 🏷️ **タグ・カテゴリ管理**: 画像/動画/チャット/コーディング
- ⭐ **お気に入り**: よく使うプロンプトをすぐに呼び出し
- 📋 **ワンクリックコピー**: プロンプトを即座にクリップボードへ
- 💾 **オフライン対応**: ローカル保存で安心

## 技術スタック

- HTML/CSS/JavaScript (Vanilla)
- IndexedDB (画像保存)
- LocalStorage (メタデータ保存)

## 開発

```bash
# 開発サーバー起動
npx serve .
```
